Agentic Workflows com GitHub Copilot

Aprenda a criar e implantar agentic workflows usando gh aw. Automatize resumos diários, importe histórias do Hacker News como issues no GitHub e configure um comando slash ChatOps — tudo pela linha de comando.

Início Rápido

Inicie a extensão CLI gh aw e configure Agentic Workflows no seu repositório em minutos.

📰

Resumos Automatizados

Crie workflows agendados que destacam as principais histórias do Hacker News como issues formatadas no GitHub todas as manhãs.

💬

Comandos Slash ChatOps

Acione agentic workflows diretamente de comentários em issues usando comandos slash — sem trocar de contexto.

Análise de Sentimento

Use IA para analisar o sentimento dos comentários de histórias do Hacker News e obtenha um resumo estruturado postado de volta no GitHub.
