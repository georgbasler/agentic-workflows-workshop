Agentic Workflows con GitHub Copilot

Aprende a crear y desplegar agentic workflows usando gh aw. Automatiza resúmenes diarios, importa historias de Hacker News como issues en GitHub y configura un comando slash ChatOps — todo desde la línea de comandos.

Lo que Aprenderás

⚡

Inicio Rápido

Inicia la extensión CLI gh aw y configura Agentic Workflows en tu repositorio en minutos.

📰

Resúmenes Automatizados

Crea workflows programados que destacan las principales historias de Hacker News como issues formateadas en GitHub cada mañana.

💬

Comandos Slash ChatOps

Activa agentic workflows directamente desde comentarios en issues usando comandos slash — sin cambiar de contexto.

🔍

Análisis de Sentimiento

Usa IA para analizar el sentimiento de los comentarios de historias de Hacker News y obtén un desglose estructurado publicado de vuelta en GitHub.
